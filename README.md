Server Log Analyzer

A Java-based tool that analyzes server logs and detects errors and warnings.

Features

- Reads server log files
- Detects ERROR messages
- Detects WARNING messages
- Generates a system health report

Technologies Used

- Java
- File Handling
- ArrayList
- Exception Handling

Sample Output

SERVER LOG REPORT

Total Logs: 6

ERRORS:
ERROR Database connection failed
ERROR Timeout connecting database

WARNINGS:
WARNING Memory usage high

STATUS: NEEDS ATTENTION
